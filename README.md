# Physiotherapy Simplification Models

This repository contains the implementation of deep learning models used for simplifying physiotherapy instructions into patient-friendly language.

## Models Included

- CNN Baseline
- LSTM Model
- T5 Transformer

## Technologies

- Python
- TensorFlow
- PyTorch
- Hugging Face Transformers

## Tasks

- Text preprocessing
- Model training
- Inference
- Evaluation

## Example

Input:
Perform cervical rotation to the right and left.

Output:
Turn your head to both sides.
